【概要】
本作品は、プログラミング学習の一環として個人で制作した、学習用Webコンテンツです。  
主に、生徒の自宅学習を支援することを目的としています。

HTML、CSS、JavaScriptを用いて制作しており、基礎文法の理解と、実際に動作するWebコンテンツの構築を通して、実践的な学習を行いました。

学習には、オンライン学習サービス「Progate」を使用し、基礎知識を身につけた上で制作を進めています。  
また、実装にあたっては、内容の理解を深めるためにChatGPTを活用しながら、処理の考え方や記述方法を確認しつつ制作しました。

【制作背景・目的】
学校から貸与されているiPadは利用制限が多く、生徒の自宅学習において十分に活用できない場面があるという課題がありました。  
そこで、生徒が私物のスマートフォンや家庭の端末から、時間や場所を選ばず学習できる環境を整えることを目的として、本Webコンテンツを制作しました。

特に、授業内で時間の都合上扱いきれなかった補足説明や詳細資料を、自宅でも確認できる点を重視しています。

【できること】
・授業内容の補足説明や資料の閲覧
・国語に関する豆知識や学習のポイントの確認
・スマートフォンや家庭用端末からの学習

【使用技術】
・HTML  
・CSS  
・JavaScript  

【工夫した点・学んだ点】
・生徒が直感的に操作できるよう、シンプルで分かりやすい構成を意識しました
・繰り返し処理や条件分岐など、JavaScriptの基礎的な記述を実装しました
・実装時にはChatGPTを参考にしつつ、処理内容を自分なりに理解し、必要に応じて書き換えや調整を行いました
・実際に利用する生徒の視点を意識し、表示内容や導線を調整しました。
・動作確認を行いながら、段階的に機能を追加・修正しました。

【個別最適化・支援の観点】
生徒の学習状況や理解度に応じて利用しやすいよう、  
個別最適化や支援的な側面を意識した設計を心がけています。

また、生徒からの要望をもとに、機能や内容を随時追加・改善し、  
利便性と学習の定着を両立させることを重視しています。

【今後改善したい点】
・コードの整理による可読性の向上
・学習内容に応じた機能の追加
・デザインやUIの改善

【補足】
本作品は学習途中の成果物であり、改善の余地がありますが、ProgateやChatGPTなどの学習支援ツールを活用しながら、課題を設定し、自分なりに考えて実装しました。
